# OPEN_DECISIONS.md

# Lyken Agency Website — Open Decisions

## Purpose

This document tracks decisions that are not fully closed yet.

It prevents the project from being blocked while keeping open items visible.

---

## 1. Decision Status Labels

Use:

- **Open** — not decided yet
- **Proposed** — recommended but not approved
- **Approved** — decided
- **Blocked** — cannot decide yet
- **Deferred** — intentionally postponed
