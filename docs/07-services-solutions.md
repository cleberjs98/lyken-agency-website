# 07 — Services & Solutions

## Lyken Agency Website

## Objetivo desta fase

A Fase 07 tem como objetivo apresentar os serviços e soluções da Lyken Agency de forma clara, comercial e premium.

A seção deve explicar o que a Lyken oferece sem quebrar a narrativa de transformação.

O objetivo é transformar a experiência conceitual em clareza comercial.

---

# 07.1 — Princípio principal

A Lyken não deve apresentar serviços como uma lista comum.

O site deve mostrar que os serviços são partes de um sistema de transformação:

```txt
estratégia → identidade → presença digital → tecnologia → crescimento
```

# 07.11 — Service card structure

Cada card deve conter:

```txt
icon
title
description
optional micro-label
hover detail
```

## Visual base

- fundo emerald green
- borda dourada fina
- ícone dourado
- texto claro
- descrição curta
- hover premium

---


```txt

```

```txt

```

Cards devem ter altura equilibrada e leitura confortável.

---

# 07.13 — Service card hover

No hover, o card pode:

- elevar levemente
- acender borda
- revelar linha dourada
- mover ícone de forma mínima
- intensificar textura

## Evitar

- giro
- bounce
- neon
- sombra pesada
- mudança de cor agressiva

# 07.18 — Copywriting rules

Os textos devem ser:

- curtos
- comerciais
- sofisticados
- objetivos
- sem exagero
- sem promessas irreais

Evitar:

```txt
We are the best
Amazing results
Explosive growth
Guaranteed success
```

Preferir:

```txt
clarity
strategy
positioning
structure
growth
digital experience
```

---

# 07.19 — Relação com a jornada

A seção de serviços deve vir após a jornada ou integrada a ela.

```txt
o que a Lyken faz na prática
```

# 07.21 — O que a Fase 07 não deve fazer

Não construir:

- formulário final
- SEO final
- deploy
- integrações externas
- animações globais finais

Foco:

```txt
serviços
soluções
diferenciais
clareza comercial
```

---

# 07.22 — Auditoria

Verificar:

- serviços estão claros?
- soluções fazem sentido?
- visual é premium?
- cards parecem Lyken?
- texto não é genérico?
- cliente entende a oferta?
- mobile está bom?

---

# 07.23 — Critérios oficiais de aprovação

A Fase 07 só pode ser aprovada quando:

- serviços principais existem
- soluções por estágio existem
- Why Lyken existe
- textos estão bilíngues
- cards estão visualmente alinhados
- layout é responsivo
- oferta comercial está clara
- visual não parece agência genérica
