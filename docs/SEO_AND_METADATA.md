# Lyken Agency Website — SEO & Metadata Specification

## Purpose

This document defines the SEO and metadata foundation for the Lyken Agency website.

The first version does not need advanced SEO, but it should be structured, shareable and ready for indexing.

---

## 1. SEO Goals

The website should communicate that Lyken Agency works with:

- branding
- brand strategy
- visual identity
- digital marketing
- websites
- web apps
- e-commerce
- AI solutions

## 6. Heading Structure

Use:

```txt
H1: Hero headline
H2: Main section titles
H3: Cards, service titles and subsections
```

Rules:

- only one H1
- headings must be meaningful
- do not use headings only for visual size

## 11. URL Strategy

First version can be one-page.

Recommended section anchors:

```txt
#about
#journey
#services
#process
#solutions
#contact
```

Future routes can include:

```txt
/about
/services
/work
/contact
```

Do not implement future routes unless approved.

---

## 12. Alt Text Rules

Use meaningful alt text for real images.

Decorative assets can be hidden or have empty alt.

Examples:

```txt
Lyken Agency monogram
Abstract gold digital mesh representing brand transformation
Lyken Agency visual identity mark
```

## 13. Semantic HTML

Use semantic elements:

```txt

section
footer

```

Avoid using divs for everything.

## 15. Performance and SEO

SEO depends on performance.

Check:

- optimized images
- no oversized textures
- no unnecessary libraries
- fast initial load
- readable content without animation dependency
