# Lyken Agency Website — Contact & Conversion Specification

## Purpose

This document defines how the contact and conversion section should work.

The goal is to turn visitor interest into a clear action.

---

## 1. Conversion Goal

Primary conversion:

```txt
Start a Project
```

The user should be able to contact Lyken clearly and confidently.

---

## 2. Contact Section Role

The contact section should:

- close the website experience
- reinforce transformation
- invite the user to start a project
- provide a simple contact form
- optionally provide direct WhatsApp contact

---

## 11. Visual Direction

The contact section should use:

- deep green background
- emerald form surface
- thin gold borders
- subtle LK watermark
- clear labels
- premium button
- generous spacing
- clean layout

Avoid:

- white generic form
- default browser inputs
- cluttered fields
- too many required fields
- poor contrast

---

## 12. Layout

```txt
Left: CTA copy and contact info
```
