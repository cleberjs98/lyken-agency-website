# Lyken Agency Website

Premium interactive bilingual website for Lyken Agency, built to support a strategic, technological, and high-end digital brand experience.

## Stack

- React
- Vite
- Tailwind CSS
- Framer Motion
- Lucide React
- JavaScript
- Vercel

## Install

```bash
npm install
```

## Run

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Preview

```bash
npm run preview
```

## Deploy

Deploy the project on Vercel after the open launch decisions are approved.

Vercel configuration:

- Framework: Vite
- Build command: `npm run build`
- Output directory: `dist`

Do not commit API keys, service tokens, `.env` files, or any other sensitive credentials to the repository.

The final domain, official email, WhatsApp, and social links are still pending in [docs/OPEN_DECISIONS.md](docs/OPEN_DECISIONS.md).
